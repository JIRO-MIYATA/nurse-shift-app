# Smart Shift Management Tool - Walkthrough

## 概要
高度な制約条件を満たすシフト表を自動生成し、ReactベースのモダンなUIで直感的に調整できるシフト管理ツールを構築しました。

## 主な機能

### 1. 自動シフト生成 (Core Algorithm)
- **3連単ルール**: 「準夜勤 → 深夜勤 → 休み」のStrictなパターンを順守して割り当てます。
- **チームバランス**: チームA・B・Cから均等に人員を配置（平日日勤は各チーム3名以上など）。
- **連続勤務制限**: 最大5連勤以下になるよう制御します。
- **スタッフ属性考慮**: リーダー必須、若手ペアリング制限、特別勤務タイプA/Bの制約を考慮。

### 2. インタラクティブな調整画面 (UI/UX)
- **Drag & Drop**: セルをドラッグして他のセルと勤務シフトを入れ替えることができます。
- **リーダーフラグ (L)**: シフトセル右上のフラグアイコンをクリックして、リーダーマークをOn/Offできます。
- **リアルタイムバリデーション**:
    - 制約違反（6連勤、不正な並びなど）が発生すると、該当セルが**赤枠**で強調表示されます。

### 3. データ出力
- **CSV Export**: 現在のシフト表をCSV形式でダウンロードできます。

## 使用方法

### 起動
```bash
npm run dev
```
ブラウザで `http://localhost:5173` にアクセスしてください。

### 操作
1. **Generate Schedule**: 画面右上のボタンを押すと、全スタッフのシフトをイチから再生成します。
2. **手動調整**:
    - 勤務を変更したいセルをドラッグして、交換先のセルにドロップします。
    - 赤枠が出た場合は、制約違反があるため修正が必要です。
3. **保存**: "Export CSV" ボタンで結果を保存します。
